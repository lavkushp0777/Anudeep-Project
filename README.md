# IPL Analysis Dashboard (2008–2025)

## Project Overview

The **IPL Analysis Dashboard (2008–2025)** is an interactive sports analytics project developed using **Microsoft Power BI** to analyze historical data from the Indian Premier League (IPL). The dashboard transforms raw match data into meaningful visual insights, allowing users to explore team performance, player statistics, and tournament trends across different IPL seasons.

This project demonstrates how **Business Intelligence (BI) tools** can be applied to sports analytics by converting complex datasets into clear, interactive dashboards and reports.

---

## Abstract

The **IPL Analysis Dashboard (2008–2025)** is an interactive data analytics project built using **Microsoft Power BI** to explore and visualize historical IPL data. The primary objective of this project is to convert raw cricket match data into meaningful insights through dynamic visualizations. This enables users to analyze team performance, player statistics, and tournament trends across multiple IPL seasons.

The dashboard provides a comprehensive overview of key IPL statistics, including **champions, runners-up, total matches played, number of teams, venues, centuries, half-centuries, and boundary statistics such as fours and sixes**. It also highlights important tournament achievements such as the **Orange Cap (highest run scorer)** and **Purple Cap (highest wicket taker)** winners, along with additional performance indicators like the players with the most fours and most sixes in a season.

An **interactive points table visualization** allows users to examine team standings based on matches played, wins, losses, and total points. By using **slicers and filters**, users can select a specific IPL season and instantly update all dashboard visuals.

Overall, the project demonstrates how **Power BI can be effectively used for data visualization, interactive reporting, and sports data analytics**, enabling deeper insights into IPL performance patterns and tournament dynamics.
